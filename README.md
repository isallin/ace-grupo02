# ⚡ Project A.C.E. — Agent Composition Engine

O **Project A.C.E.** é um motor de análise estratégica para **VALORANT**, desenvolvido como um projeto acadêmico para solucionar a complexidade da escolha de composições no cenário competitivo. A ferramenta calcula sinergias entre os 5 agentes de uma equipe para sugerir a composição com a maior probabilidade de vitória (*win rate*) baseada em dados.

---

## 🔍 O Problema
No cenário competitivo e em campeonatos de alto nível, a escolha dos agentes (o "draft") pode definir o destino de uma partida antes mesmo dela começar. Com o aumento constante do elenco de personagens, as combinações possíveis tornam-se vastas, dificultando a análise humana de qual composição é a mais eficiente para cada mapa ou situação específica.

## 💡 A Solução: A.C.E.
O **Agent Composition Engine** atua como um conselheiro tático, utilizando:
* **Cálculo de Sinergia:** Algoritmo que avalia como as habilidades de diferentes agentes se complementam.
* **Win Rate Analytics:** Análise de dados históricos para prever o desempenho de formações específicas.
* **Metodologia de 5 Agentes:** O motor não olha apenas para escolhas individuais, mas para o ecossistema completo do time.

---

## 🛠️ Estrutura do Projeto

### 1. Contextualização e Fundamentação
* Introdução ao cenário de E-sports.
* Definição das mecânicas principais do VALORANT.
* Levantamento de dados sobre o meta-jogo atual.

### 2. Metodologia
* Coleta de dados de partidas competitivas.
* Lógica de processamento do motor de composição.
* Validação estatística das sugestões do A.C.E.

### 3. Desenvolvimento Técnico
* **Frontend:** Interface web para consulta das composições.
* **Backend:** API para cálculo de sinergias em tempo real.

*Projeto desenvolvido como parte dos estudos de Análise e Desenvolvimento de Sistemas.*
